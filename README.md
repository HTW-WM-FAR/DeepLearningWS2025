# Seminar-DL

Sign Language Recognition

Description
Deep Learning project to classify American Sign Language (ASL) hand gestures using the Sign Language MNIST dataset. Two models are implemented and compared: a Convolutional Neural Network (CNN) trained from scratch and a fine-tuned Vision Transformer (ViT).

Dataset
Download the Sign Language MNIST dataset from Kaggle and place the CSV files in the same directory as the notebook:
https://www.kaggle.com/datasets/datamunge/sign-language-mnist/data
